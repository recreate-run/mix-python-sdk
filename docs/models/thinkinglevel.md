# ThinkingLevel

Thinking level: off (disabled), basic (4k tokens), medium (10k tokens), maximum (32k tokens). If not provided, determined by keywords in message.


## Values

| Name      | Value     |
| --------- | --------- |
| `OFF`     | off       |
| `BASIC`   | basic     |
| `MEDIUM`  | medium    |
| `MAXIMUM` | maximum   |
